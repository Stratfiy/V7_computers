## Pages
- Home, About, Services, Solutions, Shop (→Shopify), Contact

## Setup
1. `git clone` → open `index.html` in browser
2. Update Shopify URL in `shop.html`
3. Deploy via GitHub Pages / Vercel / Netlify

## Config
- **Shopify**: Edit `SHOPIFY_URL` in `shop.html`
- **Google Maps**: Replace iframe `src` in `contact.html`
- **Social links**: Update in footer of each page

## Tech
Pure HTML/CSS/JS — no build tools needed.
